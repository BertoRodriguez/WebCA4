Hobbies and Interests

I don't remember much from my younger years apart from that I used to love using Scratch (the programming language) I'm not sure if that's what kindled my love for coding but it would make sense.

Design Process

First I created a list of things which needed to be included: home page containing introduction, about page talking about current studies etc. I then layed out some basic semantics for the structure of the webpage, e.g. the nav bar, sections, footer etc. I then added some text to check if the layout was suitable and made changes where needed. I then chose a colour scheme and styled my webpages accordingly.
